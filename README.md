# :movie_camera: Morec - Flutter 版电影应用

这是一个非常精美的 Flutter 版电影客户端，利用豆瓣现有的 Api，打造了一个完整的电影展示 App。细节十分完善，是一个经过完整设计的产品而非 Demo。

正在完善影片详情页面以及影人详情页面，预计四月中旬全部完成。

## Set up

### 1. Clone the repo

```bash
$ git clone https://github.com/Mayandev/morec.git
$ cd morec
```

### 2. Running

```bash
flutter run
```
